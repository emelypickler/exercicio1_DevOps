# exercicio1_DevOps

## SRE

Tem como função garantir a estabilidade do sistema, evitando quedas e um bom funcionamento, eles fazem isso com a automatização de tarefas e com um monitoramento constante. 

## DevOps

Tem como função entregar o software de forma rapida e segura, cuidando da infraestrutura e usando automações para testes e atualizações.
